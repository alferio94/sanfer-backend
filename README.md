# Stack

- NestJS
- TypeORM
- PostgreSQL
- Docker
- Docker Compose
- Yarn
- Eslint

# Setup

```bash
yarn install
```

# Run

```bash
yarn run start:dev
```

# Test

```bash
yarn run test
```

# Deploy

```bash
docker-compose up -d
```
